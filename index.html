How to Upload Data
  1. Connect your Android device to the ESP32's WiFi network (Avro-Sagitta-Logger)
  2. Open the Sagitta Android app
  3. Start logging to collect data
  4. Stop logging when finished
  5. Download the CSV file from the ESP32
  6. Get your current location
  7. Press the Upload button to send data to this website
Data is automatically sent to this website when you press Upload in the Android app.

About Sagitta Logger

System Information

The Sagitta Data Logger is an ESP32-based system designed to collect temperature and pressure data along with location information.

Features:
  • Real-time temperature monitoring with MAX31865 sensor
  • Pressure sensing with ADS1115 ADC
  • GPS location tracking
  • Data logging to SD card
  • WiFi connectivity for data export
  • Low-power operation with deep sleep mode
